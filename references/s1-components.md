## Recommendations for M80

**Fastest path (smallest pools):**
1. **Eckhart Security** — 5-item pools for SonicLite (PP) + Lightstrike I-III (laser weapons)
2. **Foxwell Enforcement** — 6-7 item pools for Attrition/CF-117 (lasers), DR Model-XJ1 (distortion)
3. **Headhunters** — 6-item pool for Buzzsaw (ballistic), but 13-item for shield, 17-item for cooler

**Avoid for M80:**
- Covalex VK-00 (Master tier = 237k rep grind)
- Singe Cannon S1 (Elite Contractor = 95k rep)
