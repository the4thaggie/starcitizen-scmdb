# Rep tier screenshot notes

Use this when a user provides a Star Citizen Career/Dossier screenshot instead of a numeric rep value.

## Key takeaways
- Read the *highlighted track* on the right-hand panel for the active progression tier.
- Do not confuse the top-level faction label with the sub-track tier.
  - Example: the faction can show an overall alignment like `Covalex — Neutral` while the `Hauling` track shows `Member`.
- When the user says they are "about a third to half" to a tier, the screenshot is often better than estimating from the vague description.
- If the screen shows the tier ladder, use the highlighted tier name as the current standing and plan from there.

## Response pattern
- State the exact highlighted tier.
- Mention the parent faction label only if it helps avoid ambiguity.
- If planning a grind, use the highlighted tier as `current_tier` for mission planning.
